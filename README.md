# 📊 DashPro - Cyberpunk Slate Admin Dashboard

![Preview](screenshots/preview.png)

A dense, utilitarian admin analytics dashboard built using a flat **Slate & Cyber Orange** cyberpunk design system, breaking away from standard templates.

## ✨ Humanized & Localized Features
- 📟 **Monospace System Logs Console** — A live, simulated terminal on the dashboard printing realistic background developer logs (such as cron backup cycles and Lipa na M-Pesa STK transaction callbacks).
- 📝 **Post-It Notes Widget** — A handwritten Caveat-font sticky note section populated with Fred's actual developer sprint reminders ("Mentor meeting @ 2PM", "Fix Firefox chart overflow").
- 📊 **Visual Analytics** — Custom SVG/CSS bar charts showcasing active user metrics.

## 🛠️ Tech Stack
- **Frontend:** React, JavaScript (ES6), HTML5, CSS3
- **Libraries:** Chart.js, Fira Code (Monospace data font)
